# rn-sandbox-module

Sandbox module

## Installation

```sh
yarn add rn-sandbox-module
```

or

```sh
npm install rn-sandbox-module
```

## Usage

```js
import { SandboxLabel } from 'rn-sandbox-module';

// ...

<SandboxLabel />
```

Please see `example` folder for real example.
